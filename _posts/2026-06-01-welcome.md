Welcome to my devlog site!
As my first blog I wanted to have a little introduction to myself as well as what the hell this whole site is all about.

First, my name is Conor Damery, I've been writing code since I was 13 years old and I started with very humble beginings.
When I started learning to code, resources online were very limited and I often found it very daunting to write a piece of code. You could say for me it was akin to magic, or rather, dark magic.

Well more than decade later and the dark magic has faded away, and now I'm more of a wizard writing code like my bread and butter. However, for many people, the dark magic is still real, and I hope to shine a bit of light into some more "obscure" topics people with a basic understanding of game engines might eventually encounter.

Ok, let me pause here for a second because this is important, my devlogs are not meant to teach the very basics of coding or game engines (although perhaps every once in a blue moon), these devlogs are tailored to more in-depth topics you would learn when working on an engine, and therefore, I do not per-se recommend diving into this site if you are still struggling with the basics. Having said that, if you're up for the challenge or simply want to get your feet wet on advanced topics, by all means browse away.

Alright, back to what the plan for the devlogs is. The first series I want to explore is about this little open-source handheld I got my hands on, its called, the R36S.

![R36S]({{ site.baseurl }}{{ page.assets_dir }}/r36s.jpg)

This little device packs a punch for the price range and specs, now to be clear, it won't be running any AAA title. But by being diligent and using the right optimizations, I expect to at least run PSP quality games (as of now I've yet to see that happen!).

Alright, so that's pretty much it for this post, I want to keep it short so we can get into the next post which is actually starting to write some code for it or at least coming up with the plan on what to start coding. I hope to see you all in the next posts!
